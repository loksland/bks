bks
===

###Getting started###

```bash
$ bks
```

You will be prompted for a path to the data directory. Best to point this to a private 
repositry.

###Get help###

```bash
$ bks --help
```

###Reset user config###

```bash
$ bks --reset
```

Clear the path to the data directory and any customised directory names.

###Index data###

```bash
$ bks --index
```

Reindex all in/out data.
